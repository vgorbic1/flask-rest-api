## Activate Environment
In VS Code's GitBash:

```$ source ./venv/bin/activate```

To deactivate:

```$ deactivate```

## Start App
```
$ cd code
$ python app.py
```

## Postman
Use Python REST API Environment
Create a user with /register endpoint.
Generate JWT with /auth endpoint.
